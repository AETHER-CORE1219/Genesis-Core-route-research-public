# 発現経路研究 外部投稿パック v1

## 1. GitHub リポジトリ説明文

本リポジトリは、占星術における発現経路研究の現時点の公開固定面をまとめたものです。西洋占星術とインド占星術の両体系を用い、著名人物の出生データから反復する経路を抽出し、blind ingress と人物単位再読で検証しました。第一段階では、著名人物の表での発現が占星術的配置と反復的に一致することを経路単位で発見しました。第二段階では、その内容を数百人規模で再監査し、現時点のデータで正確に発現を記述できる部分だけを固定しました。現在の公開主張面は 16 本の固定経路であり、それ以外の保留群、science 未固定群、packet-level 候補、watch subbranch は未確定として明示的に分離しています。

## 2. GitHub 冒頭短文

著名人物の出生データを用いて、占星術的な「発現経路」を西洋+インド両体系で再監査した研究です。第一段階では表での発現と占星術的配置の一致を経路単位で発見し、第二段階ではその中で現時点でも正確に記述できる固定 16 経路だけを公開主張面として残しています。

## 3. Reddit / 掲示板向け短文

著名人物の出生データを使って、占星術的な発現経路が反復するかを西洋+インド両体系で検証しました。第一段階では表での発現と占星術的配置の一致を発見し、第二段階ではその内容を数百人規模で再監査しました。今の公開固定面は 16 経路で、他の候補は保留として分離しています。blind ingress を最終ラベル化せず、固定 / 保留 / 共通因子 / packet 候補を分けたのが今回の研究の中心です。

## 4. Reddit / 掲示板向け長文

占星術における「発現経路」が本当に反復するのかを、著名人物の出生データで検証した研究を公開します。  
この研究は二段階で進めました。第一段階では、publication 台帳 188 rows / 151 unique people を母体に、分野ごとの発現経路そのものを発見し、表での発現と占星術的配置の一致を経路単位で確認しました。第二段階では、その後の publication 外人物を現在の経路 inventory へ blind に大量投入し、数百人規模で、どの経路が太るか、どの束が共通因子か、どの候補がまだ保留かを切り分けました。  
現在の公開主張面は 16 本の固定経路です。一方で、unified 側の保留 14 本、science 側の未固定 16 本、packet-level の細い候補 1 本、watch subbranch 8 本は、まだ固定主張に入れていません。  
この公開では、固定面と未固定面を厳密に分け、blind ingress を route pressure observation として扱っています。興味がある方は full report、固定経路の二段階付録、再現 protocol から見てください。

## 5. 形式的な研究紹介文

本研究は、著名人物の出生データを対象とし、西洋占星術およびインド占星術の両体系に基づいて、職業分類以前の反復的な発現経路が存在するかを検証したものである。研究は、初期 publication cohort からの経路発見段階と、publication 外人物の blind ingress による拡張・検証段階の二段階で進めた。第一段階では、表での発現と占星術的配置の一致を経路単位で発見した。第二段階では、その内容を数百人規模で再監査し、現時点では 16 本の fixed route family / branch を公開主張面とし、underfilled 群、science 非固定群、packet-level 候補、watch subbranch は未確定として分離した。

## 7. よくある誤読への短い返答

### Q. 16経路で人間を決め打ちしているのか

いいえ。固定 16 本は route family / branch の公開主張面であり、決定論的な職業予測器ではありません。

### Q. blind ingress の top route がその人の最終分類なのか

いいえ。blind ingress は route pressure observation であり、最終ラベルではありません。

### Q. science 側の経路は無いのか

いいえ。science 側には未固定群 16 本がありますが、現時点では固定主張面に入れていません。

### Q. packet 候補や subbranch は新経路なのか

いいえ。現時点では provisional 候補または split signal です。
